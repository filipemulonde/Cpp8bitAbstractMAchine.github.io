Filipe Mulonde 8-bit computer

I'm building an 8-bit computer (turing complete) from scratch using only simple logic gates.

inspired by Ben Eater,I'm building an 8-bit computer from scratch

At the moment this computer is fully functional.This computer currently has the following modules:

  • Clock module
  • A & B Registers
  • Arithmetic logic unit (ALU)
  • Random access memory (RAM)
  • Program counter
  • instruction register
  • Output register
  • CPU control logic
  • CPU control signal
  • 8-bit decimal display
  • CPU flags register

Because of my passion for C ++ right now this computer is being updated to partially run programs written in C ++.

I am making a computer that can run the C++ described in this publication by Bjarne Stroustrupp, The full version of this computer will have hardware support for division and multiplaction, an LCD display, a floating point unit, floating point registers, interrupts and probably support for new and delete instructions. .
